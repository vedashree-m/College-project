About the Project

Our Livestock Health Monitoring System is a robust, real-time solution that integrates modern technology with traditional animal husbandry practices. The project is focused on early disease detection and proactive livestock management using wearable sensors, AI-driven predictions, and cloud integration via Firebase and Flask.

This innovative platform collects physiological parameters like temperature, heart rate, and activity levels using IoT devices. These metrics are processed in a Python-powered Flask backend and visualized for farmers using a responsive web dashboard. With Firebase Authentication, user access is secure and role-based.

Key Features:

  • Real-time health data monitoring
  • AI-based disease prediction
  • Chart.js data visualization
  • Secure login with Firebase Auth
  • Modular UI with 10 interactive pages
  • Fully mobile responsive (Tailwind CSS)

Vision and Impact

The system aims to reduce livestock mortality, improve farm productivity, and minimize economic losses caused by late diagnosis. By combining AI and IoT, it allows farmers and veterinarians to act quickly and make informed decisions.

Technology Stack

  • Frontend: HTML, TailwindCSS, JavaScript, Chart.js
  • Backend: Flask (Python)
  • Authentication: Firebase Auth
  • Hosting: Firebase Hosting / Render
  • Data: Live sensor feeds + mock data for testing

Future Plans

We plan to expand the platform by integrating mobile apps, support for voice commands, offline data sync for remote areas, and predictive models using time-series analysis and ML algorithms to improve forecast accuracy.
